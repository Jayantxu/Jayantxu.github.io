  2. 请你将两个数相加，并以相同形式返回一个表示和的链表。
  3. 你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例：

思路

两个指针，遍历链表，然后每一位相加，大于10的部分，进一位。
即个位数相加，必定是最大也就9 + 9的情况，例如9 + 8 = 17，17 % 10，余7，新的个数为7，将1进一位。

首先我们思考一下进程与线程的关系与区别：

  1. 一个进程由一个或多个线程组成，多个线程可以协同工作。
  2. 进程之间相互独立，同一进程下各个线程共享程序的内存及一些进程级的资源。
  3. 线程之间的切换比进程间的切换更快。
